HTML Reference / Cheatsheet

Isang mabilisang reference ng mga HTML tags. I-print mo para lagi mong makita!

Print-friendly HTML cheatsheet.

Document Structure

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>Page Title</title>
  </head>
  <body>
    Content here
  </body>
</html>

Text Content

TagDescriptionExample
<h1>-<h6>Headings (h1 largest)<h1>Title</h1>
<p>Paragraph<p>Text here</p>
<strong>Bold / important<strong>Bold</strong>
<em>Italic / emphasis<em>Italic</em>
<br>Line breakLine 1<br>Line 2
<hr>Horizontal rule<hr>

Links & Images

TagDescriptionExample
<a>Hyperlink<a href="url">Text</a>
<img>Image (self-closing)<img src="url" alt="desc">

Lists

TagDescriptionExample
<ul>Unordered list (bullets)<ul><li>Item</li></ul>
<ol>Ordered list (numbers)<ol><li>First</li></ol>
<li>List itemGinagamit sa loob ng ul/ol

Forms

TagDescriptionExample
<form>Form container<form action="/submit">...</form>
<input>Input field (self-closing)<input type="text">
<label>Label for input<label for="id">Name</label>
<textarea>Multi-line text input<textarea></textarea>
<button>Clickable button<button>Submit</button>
<select>Dropdown list<select><option>...</option></select>

Semantic HTML5

TagDescription
<header>Introductory content / navigation
<nav>Navigation links
<main>Primary content (use once per page)
<section>Thematic group of content
<article>Self-contained content (blog post, news)
<aside>Tangentially related content
<footer>Footer (copyright, links)
<figure>Image with caption
<figcaption>Caption for figure
