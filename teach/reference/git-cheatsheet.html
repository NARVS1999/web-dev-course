Git Cheatsheet

Config

git config --global user.name "Name"
git config --global user.email "email@example.com"

Main Workflow (Add → Commit → Push)

git init                  # start tracking a folder
git status                # check what changed
git add .                 # stage all changes
git add file.html         # stage specific file
git commit -m "message"   # save point
git push origin main      # upload to GitHub
git pull origin main      # download from GitHub

Branches

git branch name           # create branch
git checkout name         # switch to branch
git checkout -b name      # create + switch
git merge name            # merge into current
git branch -d name        # delete branch

Undo

git checkout -- file       # undo file changes
git reset HEAD file        # unstage file
git reset --soft HEAD~1    # undo commit (keep changes)
git reset --hard HEAD~1    # undo commit (delete changes)
git revert HEAD            # safe undo via new commit

Log / History

git log                   # commit history
git log --oneline         # compact history
git diff                  # see unstaged changes

Remote

git remote add origin URL  # connect to GitHub
git remote -v              # see remotes
git push -u origin main    # first push with tracking

Good Commit Messages

git commit -m "Add login form with validation"
git commit -m "Fix navbar not showing on mobile"
git commit -m "Refactor API calls into service file"
