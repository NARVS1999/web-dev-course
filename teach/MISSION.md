# Mission: Web Development for Freelancing

## Why
Gusto kong matutong gumawa ng web apps (website + mobile) para makapag-start ng freelancing career. Kailangan ko ng skills na pwedeng pagkakitaan — from simple portfolio sites to full-featured web applications na pwedeng i-run sa local machine ko nang libre.

## Success looks like
- Makakagawa ako ng To-Do App na may Laravel API, Next.js frontend, at React Native mobile — lahat tumatakbo sa local machine
- Nakapag-deploy ako ng app sa free hosting (Render + Vercel) na pwedeng ipakita sa clients
- Kaya kong gumawa ng custom websites para sa mga small business clients
- May portfolio akong maipapakita para makakuha ng freelance projects

## Constraints
- Complete beginner — walang programming experience
- Windows PC lang gamit
- Libre lahat ng tools — walang pambili ng paid services
- Taglish ang preferred language for learning
- Local development muna — sa sariling computer lang

## Out of scope
- Advanced DevOps / server administration
- Native iOS/Android development (React Expo lang)
- Paid tools or services
- Complex animations or graphic design
