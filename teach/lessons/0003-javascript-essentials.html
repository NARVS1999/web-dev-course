Lesson 0003: JavaScript Essentials

Gawing interactive ang iyong website — the magic behind the curtain!

Lesson Goal Matututunan mo ang JavaScript basics: variables, functions, DOM manipulation, events, at Fetch API. Sa dulo, gagawa ka ng interactive feature para sa portfolio mo — isang "Click to Show Message" button!

Ano ang JavaScript?

Kung ang HTML ay ang structure ng bahay at ang CSS ay ang pintura at disenyo, ang JavaScript naman ang kuryente at plumbing — nagpapagalaw at nagpapa-interact sa website.

  • HTML → "Ano ang laman?"
  • CSS → "Ano ang itsura?"
  • JavaScript → "Ano ang ginagawa nito?"

Pwede mong gamitin ang JavaScript para:

  • Mag-reply kapag nag-click ang user ng button
  • Mag-validate ng form bago i-submit
  • Kumuha ng data mula sa API
  • Gumawa ng animations at interactive features
  • At marami pang iba!

Paano Magdagdag ng JavaScript

Tatlong paraan, katulad ng CSS:

1. Inline (sa HTML tag)

<button onclick="alert('Hello!')">Click Me</button>

2. Internal (<script> sa HTML file)

<script>
  alert('Hello, World!');
</script>

3. External (recommended)

<script src="script.js"></script>
Best Practice Ilagay ang <script> sa dulo ng <body>, bago ang </body>. Para mag-load muna ang HTML bago ang JS.

Variables — Ang mga Lalagyan ng Data

// let — pwedeng magbago ang value
let name = "Juan";
name = "Pedro";  // okay lang

// const — hindi pwedeng magbago (constant)
const birthYear = 1998;
// birthYear = 1999;  // ERROR!

// var — OLD way, wag na gamitin
var oldWay = "avoid this";
KeywordPwedeng Baguhin?Kailan Gagamitin
letOoKapag magbabago ang value (e.g., counters, user input)
constHindiDefault — gamitin ito lagi unless kailangan mo ng let

Data Types

// String — text
let message = "Hello, World!";

// Number — numbers
let age = 25;
let price = 99.99;

// Boolean — true or false
let isLoggedIn = true;
let isAvailable = false;

// Array — list of items
let fruits = ["Apple", "Banana", "Mango"];
console.log(fruits[0]);  // "Apple" (zero-indexed!)

// Object — collection of key-value pairs
let person = {
  name: "Juan",
  age: 25,
  isStudent: true
};
console.log(person.name);  // "Juan"

Operators

// Arithmetic
let sum = 5 + 3;     // 8
let diff = 10 - 4;   // 6
let product = 3 * 4; // 12
let quotient = 10 / 2; // 5

// Comparison (nagbabalik ng true/false)
5 === 5    // true (strict equality — recommended!)
5 == "5"   // true (loose equality — iwasan!)
5 !== 3    // true (not equal)

// Logical
true && false  // false (AND — pareho dapat true)
true || false  // true  (OR — isa lang ok na)
!true          // false (NOT — baliktad)
Common Mistake Gamitin ang === (triple equals) hindi == (double equals). Ang === ay strict equality — sine-check pati ang data type. Ang == ay may type coercion na pwedeng mag-cause ng bugs!

Functions — Mga Reusable na Blocks ng Code

// Function declaration
function greet(name) {
  return "Hello, " + name + "!";
}

// Calling the function
let result = greet("Juan");
console.log(result);  // "Hello, Juan!"

// Arrow function (modern way!)
const greet2 = (name) => {
  return `Hello, ${name}!`;
};

// Shortcut pag isang line lang
const greet3 = (name) => `Hello, ${name}!`;
Template Literals Pansinin ang `Hello, ${name}!` — ito ang template literal (backtick, hindi quote). Pwedeng maglagay ng variables gamit ang ${} — mas malinis kaysa sa + concatenation!

DOM Manipulation — Pakikipag-interact sa HTML

Ang DOM (Document Object Model) ay ang representation ng HTML mo sa JavaScript. Gamit ito para baguhin ang content at style ng page.

// Pumili ng element
const title = document.querySelector('h1');
const button = document.getElementById('myButton');
const paragraphs = document.querySelectorAll('p');

// Baguhin ang content
title.textContent = 'Bagong Title!';
title.innerHTML = '<em>Bagong</em> Title!';

// Baguhin ang style
title.style.color = 'blue';
title.style.fontSize = '32px';

// Magdagdag ng class
title.classList.add('highlight');
title.classList.remove('old-style');
title.classList.toggle('active');  // add kung wala, remove kung meron

// Gumawa ng bagong element
const newPara = document.createElement('p');
newPara.textContent = 'Ito ay bagong paragraph!';
document.body.appendChild(newPara);

Events — Pag-react sa User Actions

<button id="myBtn">Click Me!</button>
<p id="output"></p>

<script>
  const btn = document.getElementById('myBtn');
  const output = document.getElementById('output');

  btn.addEventListener('click', () => {
    output.textContent = 'Button was clicked! 🎉';
    output.style.color = 'green';
  });
</script>

Common events:

EventKailan Nagfa-fire
clickNag-click ang user
submitNag-submit ng form
mouseoverNag-hover ang mouse
keydownNag-press ng key
loadNag-load na ang page

Conditionals — Paggawa ng Desisyon

let age = 18;

if (age >= 18) {
  console.log("Pwede nang bumoto!");
} else if (age >= 13) {
  console.log("Teenager ka pa lang.");
} else {
  console.log("Bata ka pa.");
}

// Switch — para sa maraming specific values
let day = "Monday";
switch (day) {
  case "Monday":
    console.log("Start of week!");
    break;
  case "Friday":
    console.log("Weekend na! 🎉");
    break;
  default:
    console.log("Normal day lang.");
}

Loops — Pag-uulit

// For loop — alam mo kung ilang beses
for (let i = 0; i < 5; i++) {
  console.log("Count: " + i);
}

// For...of — para sa arrays
let colors = ["red", "blue", "green"];
for (let color of colors) {
  console.log(color);
}

// Array methods (modern)
colors.forEach(color => console.log(color));

const upperColors = colors.map(c => c.toUpperCase());
// ["RED", "BLUE", "GREEN"]

Async/Await at Fetch API — Pagkuha ng Data

Ito ang ginagamit para kumuha ng data mula sa APIs (gaya ng gagawin nating Laravel API sa susunod).

// Fetch data from an API
async function getUsers() {
  try {
    const response = await fetch('https://jsonplaceholder.typicode.com/users');
    const data = await response.json();
    console.log(data);
    return data;
  } catch (error) {
    console.error('Error fetching data:', error);
  }
}

// Tawagin ang function
getUsers();
Async/Await Explained Ang async ay nagsasabi na ang function ay may asynchronous operations. Ang await ay "hintayin mo muna matapos 'to bago magpatuloy." Parang pag-order sa restaurant — mag-order ka (await), hindi ka aalis hanggang dumating ang food (response).

🛠️ Mini-Project: Add Interactivity sa Portfolio Mo

Gawin Mo Ito!

Dagdagan mo ang portfolio mo ng interactive feature — isang "Contact Me" button na magdi-display ng message kapag clinick.

Idagdag ito sa iyong portfolio.html (sa loob ng <body>):

<section>
  <h2>Get In Touch</h2>
  <p>Click the button para magpa-message!</p>
  <button id="contactBtn" style="
    padding: 12px 24px;
    background: #2563eb;
    color: white;
    border: none;
    border-radius: 6px;
    cursor: pointer;
    font-size: 16px;
  ">Click to Say Hello 👋</button>
  <p id="contactMessage" style="margin-top: 1rem; font-weight: bold;"></p>
</section>

<script>
  const btn = document.getElementById('contactBtn');
  const msg = document.getElementById('contactMessage');
  let clickCount = 0;

  btn.addEventListener('click', () => {
    clickCount++;
    const messages = [
      'Hello! Thanks for visiting! 🎉',
      'You clicked again! 😄',
      'I love making websites! 🚀',
      'Ready to build something amazing! 💪'
    ];
    const index = clickCount % messages.length;
    msg.textContent = messages[index];
  });
</script>

I-save at i-reload — mag-click ka ng button at may lalabas na message! 🎉
