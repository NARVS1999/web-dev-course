CSS Cheatsheet

Selectors

PatternMatches
elementp { } → lahat ng <p>
.class.highlight { } → class="highlight"
#id#header { } → id="header"
parent childdiv p { } → <p> inside <div>
element.classp.highlight { } → <p class="highlight">
:hovera:hover { } → kapag naka-hover

Box Model

margin: 10px;         /* labas ng border */
border: 2px solid #000; /* border mismo */
padding: 15px;         /* loob ng border */
/* content — nasa gitna */

Typography

font-family: Arial, sans-serif;
font-size: 16px;
font-weight: 700;     /* 400=normal, 700=bold */
text-align: center;   /* left, center, right, justify */
line-height: 1.6;
color: #333;

Colors

color: red;                 /* named */
color: #2563eb;             /* hex (most common) */
color: rgb(37, 99, 235);    /* RGB */
color: rgba(37, 99, 235, 0.5); /* with opacity */

Flexbox (Parent)

display: flex;
flex-direction: row;        /* row | column */
justify-content: center;    /* horizontal: start, center, end, space-between, space-around */
align-items: center;        /* vertical: start, center, end, stretch */
flex-wrap: wrap;            /* nowrap | wrap */
gap: 1rem;                  /* spacing between items */

Grid (Parent)

display: grid;
grid-template-columns: repeat(3, 1fr);  /* 3 equal columns */
grid-template-columns: 1fr 2fr 1fr;     /* 3 columns, iba-iba width */
gap: 1rem;

Responsive (Media Queries)

/* Mobile-first: default = phone styles */
@media (min-width: 768px) { /* tablet */ }
@media (min-width: 1024px) { /* desktop */ }

/* Desktop-first: default = desktop styles */
@media (max-width: 768px) { /* tablet */ }
@media (max-width: 480px) { /* phone */ }

Positioning

position: static;     /* default */
position: relative;   /* relative to normal position */
position: absolute;   /* relative to nearest positioned parent */
position: fixed;      /* relative to viewport (stays on scroll) */
position: sticky;     /* relative until scroll reaches threshold */
top/right/bottom/left: 10px;

Display

ValueBehavior
blockFull width, new line
inlineShrink-wrap, same line
inline-blockInline pero pwedeng magka width/height
flexFlexbox layout
gridGrid layout
noneHidden (removed from flow)
