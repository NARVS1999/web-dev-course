Lesson 0020: Freelancing Guide

Mula sa paggawa ng projects hanggang sa pagkakaroon ng unang client — ang iyong roadmap papuntang freelancing career.

Lesson Goal Matutunan ang praktikal na aspeto ng freelancing: pag-setup ng profiles sa platforms, paano humanap ng unang client, pag-presyo ng projects, at paano makipag-communicate ng maayos.

Part 1: Setting Up Your Profiles

Upwork

Ang Upwork ang pinakamalaking freelancing platform. Ito ang magandang simulan:

  • Gumawa ng complete profile — professional photo, headline, summary
  • Headline example: "Full-Stack Web Developer | Laravel + Next.js + React Native"
  • Ilista ang iyong skills: HTML, CSS, JavaScript, Laravel, MySQL, Next.js, React Native, Git
  • I-upload ang iyong portfolio links (Vercel, GitHub)
  • Kumuha ng Skill Certifications (may free assessments sa Upwork)

Part 2: Finding Your First Client

Strategy: Start Small

  • Family & Friends — may kakilala kang may small business? Offer kang gumawa ng website nila
  • Local Businesses — maraming small businesses sa inyo na walang website. Lapitan mo sila
  • Social Media — mag-post sa Facebook groups ng freelancing community
  • Upwork Proposals — mag-send ng 5-10 proposals per day, i-customize ang bawat isa

How to Write a Winning Proposal

Proposal Template

Subject: "I can help you with [project name]"

  1. Greeting: Address the client by name
  2. Understanding: Ipakita na na-intindihan mo ang kanilang problem (i-mention ang specific details mula sa job post)
  3. Solution: Ipaliwanag kung paano mo ito maaayos — "I will build a Laravel API with authentication, then connect it to a Next.js frontend"
  4. Experience: I-mention ang similar projects na nagawa mo (link sa portfolio)
  5. Call to Action: "I'd love to discuss this further. When are you available for a quick chat?"

How Much to Charge (Philippine Rates)

  • Beginner (0-1 year): PHP 200-500/hour or PHP 5,000-15,000/project
  • Intermediate (1-3 years): PHP 500-1,000/hour or PHP 15,000-50,000/project
  • Experienced (3+ years): PHP 1,000-2,000+/hour or PHP 50,000+/project
Don't undersell yourself Huwag masyadong mababa ang presyo — binabarat mo ang sarili mo at ang ibang freelancers. I-presyo mo ang value na binibigay mo, hindi ang oras mo.

Part 4: Client Communication

Tips for Success

  • Always over-communicate — mag-update sa client every 2-3 days kahit walang progress
  • Use simple terms — hindi lahat ng clients marunong sa tech. I-explain sa Taglish kung kinakailangan
  • Send progress screenshots — para makita ng client ang ginagawa mo
  • Set boundaries — magkaroon ng working hours at i-communicate ito sa client
  • Always deliver on time — kung maaantala, mag-advise agad (huwag maghintay sa deadline)

Part 5: From Project to Portfolio

After every project, gawin ang sumusunod:

  1. Ask for a testimonial — "Pwede po ba kayo mag-iwan ng review?"
  2. Add to portfolio — isama ang project sa iyong portfolio website na may screenshots
  3. Write a case study — anong problem, paano mo sinolve, anong tech ang ginamit
  4. Ask for referrals — "May kakilala po ba kayo na nangangailangan ng website?"

Final Task: Start Your Freelancing Journey

Gawin ang sumusunod ngayon:

  • Gumawa ng Upwork profile (kung wala pa)
  • I-update ang iyong portfolio website gamit ang lahat ng projects na nagawa mo
  • I-post ang iyong portfolio sa social media (Facebook, LinkedIn)
  • Mag-send ng 3+ proposals sa Upwork o mag-message sa 3 potential local clients
  • Itakda ang iyong rate — magsimula sa PHP 300-500/hour
