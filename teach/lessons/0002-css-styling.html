Lesson 0002: CSS Styling

Pagandahin natin ang iyong portfolio — colors, layout, at higit pa!

Lesson Goal Sa araling ito, matututunan mo kung paano mag-istilo ng web pages gamit ang CSS — colors, fonts, layout (Flexbox & Grid), at responsive design. I-aapply mo ito sa portfolio page mo para maging presentable sa potential clients!

Ano ang CSS?

Ang CSS (Cascading Style Sheets) ang nagpapaganda ng website. Kung ang HTML ay ang skeleton ng tao, ang CSS ay ang damit, pampaganda, at makeup.

  • Colors — background, text, borders
  • Typography — fonts, sizes, spacing
  • Layout — Flexbox, Grid, positioning
  • Responsive — gumaganda sa phone, tablet, at desktop

3 Ways para Magdagdag ng CSS

1. Inline CSS (sa mismong tag)

<p style="color: blue; font-size: 18px;">Blue text ito</p>

Result: Blue text ito

2. Internal CSS (sa loob ng <head>)

<head>
  <style>
    p {
      color: blue;
      font-size: 18px;
    }
  </style>
</head>

3. External CSS (recommended!)

<head>
  <link rel="stylesheet" href="styles.css">
</head>
Best Practice Gamitin ang external CSS para sa actual projects. Ganito ang ginawa natin sa ating lessons — iisang style.css lang ang ginagamit ng lahat ng lessons!

Class vs ID Gumamit ng class (.) kung uulitin mo ang style sa maraming elements. Gumamit ng ID (#) kung isa lang — gaya ng #header o #footer.

text-align: center;  /* left, center, right */

Box Model — Ang Pinaka-Importanteng Concept sa CSS

Bawat HTML element ay isang kahon. Ang Box Model ay binubuo ng 4 na layers:

/* Margin — space sa labas ng border */
margin: 10px;
margin-top: 5px;
margin: 10px 20px;  /* top/bottom left/right */

/* Border — linya sa paligid */
border: 2px solid black;
border-radius: 8px;  /* pabilog na sulok */

/* Padding — space sa loob, sa pagitan ng content at border */
padding: 15px;

/* Content — ang actual text or image */
Visual Aid Isipin mo na ang isang picture frame: ang content ay ang litrato, ang padding ay ang maputing border ng frame, ang border ay ang frame mismo, at ang margin ay ang layo ng frame sa ibang frame.

Layout: Flexbox

Ang Flexbox ang pinakamadaling paraan para mag-arrange ng elements nang pahalang o patayo. Auto-adjust ang spacing at alignment.

.container {
  display: flex;
  gap: 1rem;           /* spacing between items */
  justify-content: center;  /* horizontal alignment */
  align-items: center;      /* vertical alignment */
  flex-wrap: wrap;          /* pumunta sa susunod na linya kung walang space */
}

Layout: CSS Grid

Ang Grid naman ay para sa 2-dimensional na layout — rows at columns.

.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);  /* 3 equal columns */
  gap: 0.75rem;
}

Responsive Design

Gamit ang media queries, pwedeng mag-iba ang style depende sa screen size:

/* Default styles (desktop muna) */
body { font-size: 18px; }
.grid { grid-template-columns: repeat(3, 1fr); }

/* Tablet — kapag 768px pababa */
@media (max-width: 768px) {
  body { font-size: 16px; }
  .grid { grid-template-columns: repeat(2, 1fr); }
}

/* Phone — kapag 480px pababa */
@media (max-width: 480px) {
  .grid { grid-template-columns: 1fr; }
}
Mobile-First Sa modernong web dev, mobile-first ang approach — mag-design ka muna para sa phone, saka mo i-adjust para sa tablet at desktop gamit ang min-width sa media queries. Pero okay lang na mag-umpisa sa desktop kung beginner ka pa lang!

Tailwind CSS (Introduction)

Ang Tailwind CSS ay isang "utility-first" CSS framework. Sa halip na magsulat ng custom CSS, gagamit ka ng classes sa HTML mo: 

<!-- Regular CSS approach -->
<div class="card">...</div>
<style>.card { background: white; padding: 1rem; border-radius: 8px; }</style>

<!-- Tailwind approach (no custom CSS needed) -->
<div class="bg-white p-4 rounded-lg">...</div>

Sa ngayon, mag-focus muna tayo sa pure CSS para maintindihan mo ang fundamentals. Darating din ang Tailwind sa advanced lessons!

🎯 Lesson Summary

  • Ang CSS ang nagpapaganda ng HTML — colors, layout, typography
  • 3 ways to add CSS: inline, internal, external (recommended ang external)
  • Selectors: element (p), class (.class), ID (#id)
  • Box Model: content → padding → border → margin
  • Flexbox para sa 1D layouts (row or column)
  • Grid para sa 2D layouts (rows + columns)
  • Media queries para sa responsive design
  • Na-istilo mo na ang portfolio mo! 🎨
