# Teaching Notes

## User Preferences
- **Language:** Taglish (Tagalog + English mix)
- **Level:** Complete beginner — no programming experience
- **OS:** Windows
- **Budget:** Zero — libre lahat ng tools
- **Goal:** Freelancing — practical, portfolio-building projects
- **Style:** Local-first — everything runs on local machine
- **Tone:** Encouraging, patient, with analogies
- **Format:** Lessons should be self-contained HTML files that print well

## Style Guide
- Explain concepts in simple terms with real-world analogies
- Use Filipino/Taglish for explanations, English for code and technical terms
- Always show complete, working code examples
- Each lesson should end with a mini-project or exercise
- Include interactive quizzes for skill reinforcement
- Reference high-quality free resources (MDN, JavaScript.info, etc.)
